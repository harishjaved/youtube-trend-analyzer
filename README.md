# 📺 YouTube Trend Analyzer

A Streamlit-based dashboard to analyze trending YouTube videos using the YouTube Data API.

## 🚀 Features
- Trending video analysis by category
- Custom search functionality
- Engagement rate calculation
- Channel leaderboard
- Interactive visualizations

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- YouTube Data API

## ▶️ Run

```bash
pip install -r requirements.txt
streamlit run youtube_dashboard.py
